## 介绍

![Demo](screenshot/upan_auto_copy_v1.0.0.gif)

upan_auto_copy是一款跨平台的**U盘自动拷贝**稳定性测试脚本工具。

## ✨特点
- 使用简单，自带中文分词、拼音、补全
- 毫秒级搜索响应
- 低CPU、内存资源占用
- 轻量安装包
- 简单大方

## 编译 
- 搭建Tauri开发环境 (https://tauri.studio/docs/getting-started/setting-up-macos)
- 运行 `yarn`
- 运行 `yarn build`
- 运行 `yarn tauri-build`
