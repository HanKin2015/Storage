# 使用大量构造密码库破解rar文件

test.rar密码是hj58

## 难受的错误：File CRC Error
```
file.extractall(path=output_path, pwd=pwd)
```
上面这句代码在密码错误的时候会抛出异常，File CRC Error。
百度找不到错误原因，有说rar文件损坏或者不完整。

后来意识到应该是密码错误导致，把它放在try语句中即可。
