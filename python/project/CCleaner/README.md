# 垃圾文件清理器

临时文件（如：.tmp、._mp）、日志文件（.log）、临时帮助文件（.gid）、磁盘检查文件（.chk）、临时备份文件（如：.old、*.bak）以及其他临时文件。

## 清理目标
系统盘 %system% 下文件类型：
```
【临时文件（*.tmp）】
【临时文件（*._mp）】
【日志文件（*.log）】
【临时帮助文件（*.gid）】
【磁盘检查文件（*.chk）】
【临时备份文件（*.old）】
【Excel备份文件（*.xlk）】
【临时备份文件（*.bak）】
```

用户目录 %userprofile% 下文件夹
```
【COOKIE】 cookies\*.*
【文件使用记录】 recent\*.*
【IE临时文件】 Temporary Internet Files\*.*
【临时文件文件夹】 Temp\*.*.
```

Windows 目录 %windir% 下文件夹
```
【预读取数据文件夹】 prefetch\*.*
【临时文件】 temp\*.*
```
