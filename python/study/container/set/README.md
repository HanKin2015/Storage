# Python集合Set中add与update区别介绍
set.add()方法用于向集合中添加单个元素。
```
>>> s = set([1, 2, 3])
>>> s.add(4)
>>> s
{1, 2, 3, 4}
>>> s.add(3)
>>> s
{1, 2, 3, 4}
```

set.update()方法用于向集合中添加多个元素，可以接受其他集合或列表作为参数。
```
>>> s = set([1, 2, 3])
>>> s.update([4, 5, 6])
>>> s
{1, 2, 3, 4, 5, 6}
>>> s.update({7, 8, 9})
>>> s
{1, 2, 3, 4, 5, 6, 7, 8, 9}
```
