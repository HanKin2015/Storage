RAII是一种C++编程技术，全称为Resource Acquisition Is Initialization，中文翻译为“资源获取即初始化”。它的核心思想是：在对象的构造函数中获取资源，在对象的析构函数中释放资源。这种技术可以确保资源的正确获取和释放，避免了资源泄漏和错误使用的问题。RAII技术在C++中广泛应用，可以用于管理各种资源，如内存、文件、锁等。
